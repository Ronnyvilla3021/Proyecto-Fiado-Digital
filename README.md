# 🥇 Fiado Digital

Sistema de gestión de créditos ("fiados") para tiendas pequeñas. Permite controlar ventas, clientes, créditos y pagos que tradicionalmente se manejan en cuadernos o chats, con dashboard en tiempo real y automatización de cobros.

## 🚧 Estado del proyecto
En desarrollo activo.

## 🛠️ Stack
- **Frontend:** Vue 3 + Vite
- **Backend:** Node.js + Express
- **Base de datos:** PostgreSQL
- **Tiempo real:** Socket.io

## 📋 Módulos
- Autenticación y roles (Admin, Cajero, Supervisor)
- Gestión de clientes
- Ventas (contado / fiado)
- Créditos y pagos
- Dashboard con gráficos en tiempo real
- Automatización de notificaciones (cron jobs)
- Reportes exportables (PDF / Excel)
